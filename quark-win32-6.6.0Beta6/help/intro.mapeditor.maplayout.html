 1.5.9. Map layout

  [ Prev - Up - Next ] 

These are the different map-layouts you can work in. Some might be easier to understand than others, but thats up to you to choose one you like. You will always be able to switch between layouts when you're working. See 'Layout menu' on how you change between these.

However, you should be aware that rotating the 'Compass and Zoom' will affect all 2D-views, so the X,Y,Z axises will correspond to the angle of the compass. If you use the 4-views layouts, you should not need to change the angle of the compass.
